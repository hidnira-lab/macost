# Quick Task 260704-oux: Fix Tauri session persistence via capabilities file Summary

**Root-caused and fixed the Tauri desktop "must log in again after every restart" bug: the project had zero Tauri v2 capabilities files, so tauri-plugin-store's get/set/save/delete calls (used by apps/web/lib/auth/session.ts to persist the JWT) were being silently rejected by Tauri's permission system, and `.session.dat` was never written to disk.**

## Performance

- **Duration:** ~12 min
- **Tasks:** 1 automated + 1 pending manual verification
- **Files modified:** 2 (1 created, 1 auto-regenerated)

## Accomplishments
- Created `apps/native/src-tauri/capabilities/default.json` granting `core:default` + `store:default` to the `"main"` window
- Confirmed both permission identifiers are exact, valid `const` values in `apps/native/src-tauri/gen/schemas/desktop-schema.json` (not guessed) before writing the file
- Verified via `cargo check --manifest-path apps/native/src-tauri/Cargo.toml` — this exercises Tauri's real build-time ACL resolver (`tauri_build::build()`), not just JSON well-formedness — build finished cleanly with no permission/ACL errors
- `apps/native/src-tauri/gen/schemas/capabilities.json` auto-regenerated as an expected build side-effect, committed alongside

## Task Commits

1. **Task 1: Add capabilities file granting store plugin access** - `b7e9f3c` (fix)

## Files Created/Modified
- `apps/native/src-tauri/capabilities/default.json` - New file, grants `core:default` + `store:default` to window `"main"`
- `apps/native/src-tauri/gen/schemas/capabilities.json` - Auto-regenerated by Tauri's build-time codegen to reflect the new ACL; not a manual edit

## Decisions Made
- No existing file touched (`tauri.conf.json`, `Cargo.toml`, `lib.rs`, `session.ts` all unchanged) — this was purely a missing-configuration fix, not a code change
- Used the plugin's bundled `:default` permission set rather than enumerating individual `store:allow-*` permissions, since `session.ts` uses the full get/set/save/delete/load surface anyway
